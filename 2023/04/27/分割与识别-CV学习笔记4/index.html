    分割与识别-CV学习笔记4

    发表于 2023-04-27 分类于 学习笔记

    概述：纹理表示和图像分割是计算机视觉领域的重要概念。纹理表示涉及图像中重复出现的模式，用于表征材料特性和外观信息，而图像分割则旨在将图像分成具有相似性质的部分。此外，在视觉识别领域，词袋模型是图像表示的一种重要方法。

    纹理表示

    纹理texture的一个简单的定义为： 一个模式或多个模式在一个区域上的重复出现，是图像是重要特性。模式是重复出现的，但模式具有变异性。形状、大小、方向、颜色具有随机性。

    纹理通常表明了一种材料的性质。同时是重要的外观信息，特别是当形状是相似的物体之间，可以帮助我们区分形状、边界等。因此我们需要找到一个比滤波(filters)和边(edges)更高级的特征表示来图像的纹理。

    纹理可分为规则纹理和随机纹理。

    基于卷积核组的纹理表示方法

    主要思想：利用卷积核组提取图像中的纹理基 + 利用基元的统计信息来表示图像中的纹理。

    流程：

    1. 设计卷积核组
    2. 利用卷积核组对图像进行卷积操作获得对应的特征响应图组
    3. 利用特征响应图的某种统计信息来表示图像中的纹理

    常见的卷积核组，包括对不同尺度、不同方向和提取不同图像信息的卷积核。如下图所示：

    纹理表示的应用

    • 纹理分类：纹理特征+聚类
    • 图像检索：纹理特征+相似度衡量
    • 场景分类
    • 图像分割

    图像分割

    概述

    目标：就是将具有相似性质的像素分在一起。

    图像分割的分类
    • 语义分割：语义分割是对图像中的每个像素都划分出对应的类别，即实现像素级别的分类。
    • 实例分割：实例，即为类的具体对象，实例分割不但要进行像素级别的分类，还需在具体的类别基础上区别开不同的个体。
    分割方式
    • “Bottom-up" 自底向上：通过分析像素底层的相似性去连接。
    • “top-down”自顶向下：凭借语义的相似性把图像进行分割。
    gestalt理论

    1.人类在自然界中的视觉是基于群组感受的。从这个角度来考虑，人是从整体来考虑这个事物的。

    2.集合中的元素可以具有由关系产生的属性。

    gestalt影响因素：临近性、相似性、同命运（方向）、同区域、平行、对称、相交、闭合

    基于聚类的图像分割

    K-means聚类

    方法

    基于强度或颜色的K-means聚类本质上是图像属性的矢量量化。聚类不需要是空间相干的。

    为了增强像素聚类效果，通过增加特征维度来实现，除了基于颜色空间，还可以添加物理空间信息，这种方法可以使得图像分割更加精准但是同时也会带来更多的过分割情况。

    优缺点分析

    优点：

    • 收敛到误差函数的局部最小值

    缺点：

    • 需要手动选择k
    • 对初始化方式很敏感
    • 对外点很敏感
    • 基于一个球形假设。（非球形假设：使用高斯混合模型，只要符合正态分布即可，但计算量比K-means大）

    Mean-shift

    方法

    Mean-shift算法的主要思想：在特征空间中去寻找模式或者局部最大密度。以下面这张图举例，假如要找到这些散布点的重心位置，使用一个较小的窗口去比较其中心和重心位置，通过重心位置的漂移进行调整。经过不断调整，直到漂移结束，就能找到密度最大的那个点。

    算法流程
    1. Find features（color，gradient，texture，etc）
    2. Initialize window at individual feature points
    3. Perform mean shift for each window until convergence
    4. Merge window that end up near the same “peak” or mode
    优缺点分析

    优点

    • 不需要球形假设
    • 可以找到可变数量的类（对于不同的window_size)
    • 对于噪点稳健

    缺点

    • 输出结果依赖于窗口大小的选择
    • 计算代价有点高（改进：对于一个点来说，其附近范围内的点mean-shift的结果相同，因此可以一起计算）
    • 对于高维的特征可能不那么友好

    每个节点代表一个像素，边表示这对像素是具有相似性的，边上的权值就是相似性的描述。通过这样的一种描述，对整幅图像在这个基础上进行分割。

    分割的依据就是切割的那些边的权值的和最小。意思就是相似性最小的那些点被分割开来了。

    Graph cut

    图割：去除边的集合使图断开成若干部分

    cut cost：切割边的权值的总和。一个好的分割即cut cost最小。（minimum cut算法） 图割算法存在的弊端就是，容易分割出比较小的区域或孤立的点。所以这种标准的图割通常会产生很多细小的区域，因此一般不是很常用。基于此的改进的是一种叫做归一化的图割方法。

    视觉识别

    主要视觉识别任务

    • 分类图像或视频
    • 检测和定位对象
    • 估计语义和几何属性
    • 对人类活动和事件进行分类

    视觉识别难点

    • 物体类别多
    • 同一问题不同视角成像问题
    • 光照
    • 尺度（同一物体再不同图片的尺度问题）
    • 形变
    • 遮挡
    • 背景杂波
    • 同类物体的差异

    视觉识别基本问题

    表示

    图像表示成向量

    • 用区域、局部特征表达图像
    • 词袋模型

    图像分类策略

    • 判别式模型（建模后验）：近邻分类、SVM、boosting、神经网络

      对于判别式模型来说求得P(Y|X)，对未见示例X，根据P(Y|X)可以求得标记Y，即可以直接判别出来。即判别式模型是对条件概率建模，学习不同类别之间的最优边界，无法反映训练数据本身的特性，能力有限，其只能告诉我们分类的类别。

    • 生成式模型（建模似然和先验）：朴素贝叶斯、LDA、隐马尔可夫模型HMM

      生成式模型求得P(Y,X)，对于未见示例X，先求出X与不同标记之间的联合概率分布，再由贝叶斯公式计算得到条件概率，取条件概率大的作为结果Y。即生成式模型对每一个类建立一个模型，有多少个类别，就建立多少个模型。

    学习

    识别

    词袋模型

    词袋模型起源

    纹理表示：每个图有不同种类的纹理，统计每个纹理的直方图（出现数量），右边这就是词袋。

    文档：统计文档中每个单词出现了多少频率，这样每个文档就能用一个（二维）向量表示。

    词袋模型构建

    提取特征

    特征提取，可以使用规则区域，或者SIFT方法。

    构建“视觉词典”

    将所有SIFT特征投影到高维空间，做聚类。聚类完后，每个聚类中心作为一个“视觉词汇”，聚类结果的个数就是“单词”的总个数。

    （注意：每个SIFT特征，都是128维度向量表示，假设数据集1000张图片，总共提取到5万个SIFT特征，那么就是对这个5万个128维度向量进行聚类，聚类可以使用k-means）

    用“视觉词汇”的频率来（向量化）表示图像

    用得到的“视觉词汇”去对每一张图像进行向量化表达。

    词袋模型的问题

    1. 词汇太多、太少都不好。

    2. 计算效率问题（K-means聚类很慢）

    3. 顺序、位置问题

      • 比如：“我爱中国”与“爱我中国”，字都一样但含义不一样。

      • 解决办法：使用空间金字塔

        1. 在用“词汇”频率直方图描述图像时，选择3种尺度。假设下图有100个特征，level0就是100维，共100个值。level1是把图像4等分切，单独计算每一块直方图，这就有4个直返图，共400个值，level2是16等分切原图，得到16个直方图，共1600个值。最后，用100+400+1600=2100个值来表示这个图像。
        2. 这种图像向量化方法，能够稍微提升图像分类时的精度。
