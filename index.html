多模态融合(Multimodal Fusion)论文总结

发表于 2024-05-13 分类于 论文详解

多模态融合是将来自多个模态的信息集成在一起，以预测一个结果度量的概念。多模态融合可以带来三个主要好处：（1）融合同一现象的多个模态，可能会导致更强大的预测，（2）捕捉多个模态之间的互补信息，（3）在缺失模态情况下，多模态系统仍然运行。多模态融合具有非常广泛的应用范围，包括音频-视觉语音识别、多模态情感识别和多媒体事件检测。

多模态表征(Multimodal Representation)论文总结

发表于 2024-04-19 分类于 论文详解

多模态表示是利用来自多个这类实体的信息来表示数据的方法。表示多种模式的数据具有更多挑战性：如何结合来自异构来源的数据；如何处理不同级别的噪声。良好的表示对于模型的性能至关重要，一个良好的表示应该具备平滑性、时间和空间相干性和自然聚类等性质。同时，多模态表示的理想属性还应该包括表示空间中的相似性反映相应概念的相似性，在某些模态的缺失的情况下表示也应该很容易获得，可以根据观察到的模态表示填充缺失的模态表示。

PEFT(Parameter-Efficient Fine-Tuning)方法总结

发表于 2024-03-10 分类于 论文详解

PEFT 技术旨在通过最小化微调参数的数量和计算复杂度，来提高预训练模型在新任务上的性能，从而缓解大型预训练模型的训练成本。这样一来，即使计算资源受限，也可以利用预训练模型的知识来迅速适应新任务，实现高效的迁移学习。因此，PEFT 技术可以在提高模型效果的同时，大大缩短模型训练时间和计算成本。

多模态情感分析论文总结

发表于 2024-01-14 分类于 论文详解

CLIP及其改进工作总结

发表于 2023-10-14 分类于 论文详解

CLIP（Contrastive Language-Image Pre-training）是由OpenAI开发的深度学习模型，用于将文本和图像之间的关系建模。该模型通过对比学习原理，将相关的文本和图像嵌入到相邻点，而将不相关的文本和图像嵌入到远离点，从而实现对图像和文本的理解。CLIP适用于多种任务，具有良好的可解释性，广泛用于图像搜索、图像标注、虚拟现实等领域。它代表了深度学习领域中在文本和图像之间建立联系的一项重要进展。
